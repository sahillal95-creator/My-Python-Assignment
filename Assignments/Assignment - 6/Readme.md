# Simple Calculator Using Tkinter

## Project Description
This is a simple calculator built using Python and Tkinter.
The calculator performs basic arithmetic operations.
All calculations are done manually using global variables and conditional statements.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal support
- Clear (C) button
- Division by zero error handling

## How It Works
1. The user enters the first number.
2. When an operator (+, -, *, /) is pressed, the first number and operation are stored.
3. The user enters the second number.
4. When "=" is pressed, the program performs the calculation manually and shows the result.

## How to Run
1. Make sure Python 3 is installed.
2. Save the file as calculator.py
3. Open terminal or command prompt.
4. Run the command: python calculator.py
5. The calculator window will open.
