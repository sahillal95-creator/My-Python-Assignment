## Task 1: Factorial Using Recursion

This program defines a recursive function named `factorial()` that:

- Takes a non-negative integer as input
- Uses recursion to calculate the factorial
- Includes proper base cases:
  - factorial(0) = 1
  - factorial(1) = 1
- Handles invalid input using try-except
- Displays the calculated factorial

### Example:
Input: 5  
Output: Factorial of 5 is 120
