# Assignment - 3 

## Task 1: Factorial Using Function
This program defines a function named `factorial()` that:
- Takes a number as input
- Calculates factorial using a loop
- Returns the factorial value
- Prints the result

Example:
Factorial of 5 is 120

## Task 2: Using Math Module
This program:
- Takes user input
- Uses Python's built-in `math` module
- Calculates:
  - Square root
  - Natural logarithm (log base e)
  - Sine (in radians)
- Displays the results

Example:
Input: 25  
Square Root of 25.0 = 5.0  
Natural Log of 25.0 = 3.2188758248682006  
Sine of 25.0 = -0.13235175009777303
